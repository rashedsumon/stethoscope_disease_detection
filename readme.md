# Digital Stethoscope: Heart and Lung Sound Analysis

This project aims to build an advanced machine learning and generative AI model capable of analyzing heart and lung sounds from digital stethoscope recordings. The objective is to separate heart and lung sounds and detect cardiovascular and pulmonary diseases.

## Requirements

- Python 3.11.0
- Streamlit for app development
- KaggleHub for dataset downloads
- Librosa for audio processing
- TensorFlow for neural networks
